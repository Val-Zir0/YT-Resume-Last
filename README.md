# ✨ Features

✅ Auto Resume

Automatically opens the last played video or music.

---

✅ Restore Playback Position

Continue from the exact second you stopped.

---

✅ Restore Playlist

Resume playlists without losing your position.

---

✅ Restore Playback Speed

Playback speed is restored automatically.

---

✅ Restore Volume

Volume level is restored.

---

✅ Restore Mute

Mute state is restored.

---

✅ Restore Shuffle

Supported on YouTube Music.

---

✅ Restore Repeat

Supported on YouTube Music.

---

✅ Mini Player

Automatically switch to Mini Player after resuming.

---

✅ Smart Homepage Redirect

Opening

```
youtube.com
```

or

```
music.youtube.com
```

takes you directly to your last session.

No homepage flash.

---

✅ Minimum Watch Time

Avoid saving accidentally opened videos.

Default:

```
30 Seconds
```

---

✅ Granular Settings

Enable or disable every feature independently.

# ❤️ Why YT Resume Last?

Unlike most extensions,

YT Resume Last restores

- Playback Position
- Playback Speed
- Volume
- Mute
- Playlist
- Shuffle
- Repeat
- Mini Player

making your YouTube session feel uninterrupted.

# 🔒 Privacy

This extension

✅ Does NOT collect personal information.

✅ Does NOT send any data to external servers.

✅ Does NOT track browsing activity.

All settings and playback information are stored locally using

```
chrome.storage.local
```
